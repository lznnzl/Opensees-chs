=====================
 Bcast command
=====================

.. function:: Bcast(data)

   Broadcast information from processor 0 to all processors.

   ===================================   ===========================================================================
   ``data`` |listi|                      can be a list of integers
   ``data`` |listf|                      can be a list of floats
   ``data`` |str|                        can be a string
   ===================================   ===========================================================================
